KLRD Site considerations

1. Rebuild site to better serve mobile devices.

Site should "respond" to the user's screen size, and make all menus available without zooming.

 

2. Change layout of site to clarify order of importance.

Change from three major content columns to two columns - one for sub-menu, and one for content.

 

3. Reduce reliance on PDFs.

HTML has more accessibility features than PDFs for screen reading.

 

4. Improve search feature, and search results.

Add filtering and sorting options; use categories and tags to organize content.

 

5. Move to a template-based website or CMS.

Manage content, encourage multiple writers and editors, democratize the website process.
